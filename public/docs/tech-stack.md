Use Next.js App Router + TypeScript + Tailwind + shadcn/ui. Prefer shadcn components for Card, Button, Separator, Badge, Tabs, Select, and Input.

### Site
- Next.js App Router (static-first)

### Dependencies
- Tailwind
    - CSS utilities and spacing system
- shadcn/ui
    - components:
        - Button
        - Card
        - Badge (chips)
        - Separator
        - Tabs
        - Select
        - Input
- Embla Carousel
    - Subtle logo strip carousel
    - Publication / Report carousel
- Lucide
    - Clean, consistent monochrome icons
- clsx
    - Conditional class name handling
- react-countup
    - One-time count-up animations for stats (no looping)
- date-fns
    - Sorting and filtering by date
